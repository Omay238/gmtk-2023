Bug-fixing simulator

This is my submission for the GMTK Game Jam 2023. The theme was "Roles Reversed," where you had to make the player play the part that they don't normally do. To play, you need some JavaScript knowledge, but not much experience. On the left, you edit the game code, on the right, you see the game window, and on the bottom right, you see the console.
